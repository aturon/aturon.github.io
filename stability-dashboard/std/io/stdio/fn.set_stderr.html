Function std::io::stdio::set_stderrExperimental[src]

pub fn set_stderr(stderr: Box<Writer + Send>) -> Option<Box<Writer + Send>>

Resets the task-local stderr handle to the specified writer

This will replace the current task's stderr handle, returning the old handle. Currently, the stderr handle is used for printing failure messages during task failure.

Note that this does not need to be called for all new tasks; the default output handle is to the process's stderr stream.
