The Fisher F distribution F(m, n).

This distribution is equivalent to the ratio of two normalised chi-squared distributions, that is, F(m,n) = (χ²(m)/m) / (χ²(n)/n).

Example

fn main() { use std::rand; use std::rand::distributions::{FisherF, IndependentSample}; let f = FisherF::new(2.0, 32.0); let v = f.ind_sample(&mut rand::task_rng()); println!("{} is from an F(2, 32) distribution", v) }
