pub unsafe fn deallocate(ptr: *mut u8, size: uint, align: uint)

Deallocate the memory referenced by ptr.

The ptr parameter must not be null.

The size and align parameters are the parameters that were used to create the allocation referenced by ptr. The size parameter may also be the value returned by usable_size for the requested size.
