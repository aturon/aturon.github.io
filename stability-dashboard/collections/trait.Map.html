A map is a key-value store where values may be looked up by their keys. This trait provides basic operations to operate on these stores.

Required Methods

fn find<'a>(&'a self, key: &K) -> Option<&'a V>

Return a reference to the value corresponding to the key.

Example

extern crate collections; fn main() { use std::collections::HashMap; let mut map = HashMap::new(); map.insert("a", 1i); assert_eq!(map.find(&"a"), Some(&1i)); assert_eq!(map.find(&"b"), None); }
use std::collections::HashMap;

let mut map = HashMap::new();
map.insert("a", 1i);
assert_eq!(map.find(&"a"), Some(&1i));
assert_eq!(map.find(&"b"), None);

Provided Methods

fn contains_key(&self, key: &K) -> bool

Return true if the map contains a value for the specified key.
