Enum core::fmt::FormatErrorExperimental[src]

pub enum FormatError {
    WriteError,
}

The error type which is returned from formatting a message into a stream.

This type does not support transmission of an error other than that an error occurred. Any extra information must be arranged to be transmitted through some other means.

Variants

WriteError

A generic write error occurred during formatting, no other information is transmitted via this variant.
